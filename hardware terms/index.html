Input Device

An input device is any hardware device that sends data to a computer, allowing you to interact with and control it. Keyboards, Buttons on phones, and the mouse for your computer are all input devices.

Output Device

Is a device that turns 0's and 1's into physical products or actions. Speakers, printers, and screens are all output devices.

Central Processing Unit (CPU)

CPU is divided into 2 parts: an ALU, and the control unit. It retrievs and exectutes instuctions. The CPU is the brain of the computer.

Arithmetic Logic Unit (ALU)

An ALU is a integrated circuit within a CPU or GPU that performs arithmetic and logic operations. its responsible for addition, subration, multiplication and division.

Control Unit

The control unit is responsible for logic and e3xecuting instructions. It tells the computer's memory, arithmetic logic unit and input and output devices how to respond to the instructions that have been sent to the processor.

Random Access Memory

RAM is the short term memory in your computer where data is stored as the processor needs it. RAM is extremly fast. it stores the information that your compouter is activly and frequently using so it can be accessed quickly.

Optical Memory

Optical memory is an electronic storage medium that uses a laser beam to store and retrieve the data. it is used with CD's and DVD's.

Magnetic Memory

Magnetic Memory is what is used to store information on devices like hardrives. It is the storage of data on a magnetized meduim.

Flash Memory

Flash memory is a memory storage medium that can be electrically erased and reprogrammed. The two main types of flash memory are NOR flash and NAND flash. It is commonly used in portable electronics and removable storage devices.

Logic Gates

Logic gates are the basic building blocks of any digital system. They are electronic circuits that have one, or more than one input and only one output.
